📊 Customer Shopping Behavior Analysis
🔍 Overview
This project focuses on analyzing customer shopping behavior using data analytics techniques. It involves data loading, cleaning, exploratory data analysis (EDA), SQL querying, and visualization using Power BI. The goal is to extract meaningful insights that can help businesses make data-driven decisions.
📁 Dataset
The dataset contains customer transaction and behavior data.
Includes features like:
Customer ID
Age
Gender
Purchase Amount
Product Category
Purchase Frequency
Payment Method
Data was loaded and processed using Python.
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI (Dashboard creation)
Gamma (Presentation creation)
Jupyter Notebook / VS Code
⚙️ Project Steps
1. Data Loading
Imported dataset using Pandas
Checked structure, data types, and basic info
2. Data Cleaning
Handled missing values
Removed duplicates
Corrected inconsistent data
Converted data types where necessary
3. Exploratory Data Analysis (EDA)
Analyzed customer demographics
Studied purchase patterns
Visualized trends using graphs and charts
4. SQL Analysis
Imported cleaned data into SQL database
Performed queries like:
Total sales by category
Customer segmentation
Frequency analysis
Used joins, aggregations, and filtering
5. Dashboard Creation (Power BI)
Built interactive dashboard
Included:
Sales trends
Customer segmentation
Product performance
Used filters and slicers for better insights
6. Report & Presentation
Created a detailed report summarizing findings
Designed PPT using Gamma for presentation
